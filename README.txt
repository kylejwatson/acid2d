A simple 2D game engine that will support automated rendering and updating the 
<canvas>. Eventually I want the engine to have a working physics engine and 
maybe some image manipulation/pixel shaders.
-------------------------------------------------------------------------------
Installation instructions

Place acid.js before your main game code. If you want to place the main game 
canvas where you want on your page, put it wherever with any attributes but 
make sure to change the id to 'main'. If not the 'main' canvas will be created 
at the bottom of the page with dimensions 800x600.
